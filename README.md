## Installation:

To get started with this project, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/themrinmoy/shop_project.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

## Usage:
- **start:**
